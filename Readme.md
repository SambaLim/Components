## 🤷‍♂️ 어떻게 사용해야 할까요?

각 컴포넌트 별로 사용방법이 다를 수 있습니다. 따라서 **예시페이지**(`index.html`)를 제공합니다.

### 📂 디렉토리

만든 컴포넌트(`{Components}`)들을 각 폴더별로 분리하여 넣었습니다.
각 컴포넌트는 예시 페이지(`demo/index.html`)가 있어 이를 보고 사용하실 수 있습니다.

```
│
├── / {Components}
│   ├── / demo
│   │   ├── index.html
│   │   └── {Components}.css
│   ├── {Components}.js
│   └── Readme.md
│
└── Readme.md
```
